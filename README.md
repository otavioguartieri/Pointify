# Pointify

Este projeto é um sistema de gamificação baseado em pontos que transforma disciplina e consistência em recompensas reais.

A proposta é simples: cada ação concluída — como treinar, estudar, cumprir metas pessoais ou manter hábitos positivos — gera pontos. Esses pontos podem ser acumulados e trocados por recompensas previamente definidas pelo próprio usuário.

O objetivo é criar um ciclo positivo de motivação, onde esforço gera progresso visível, e progresso gera recompensa.

Diferente de aplicativos tradicionais de metas, este sistema coloca o usuário no controle total das regras, valores de pontuação e recompensas, tornando a experiência personalizada e adaptável para qualquer área da vida: saúde, estudos, produtividade, finanças ou desenvolvimento pessoal.
